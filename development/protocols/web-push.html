Web Push Notifications

Web Push notification payload is a JSON-encoded structure that describes a new or updated story or deletion of one of the stories. The structure is described below.

STORY_ADDED

Story added or updated.

Type Field Comment
String type story-added
String originUrl URL of the node the notification is coming from
StoryInfo story the story

STORY_DELETED

Story deleted.

Type Field Comment
String type story-deleted
String originUrl URL of the node the notification is coming from
String id ID of the story to be deleted
